# CookieClicker
*Minecraft Version: 1.8*  
**CookieClicker** is a fun minigame where you click to get Cookies, buy upgrades and try to get as many Cookies as you can.**

## Gameplay  

- Click the Block to start the game
- Click to gain more Cookies
- Upgrade your cookies per click
- Try to get as many Cookies as you can

## Commands
- /cc *Shows all commands*
- /cc perms *Shows all permissions*
- /cc getclicker *Gives you a new Cookie Block*

## Installation  

1. Download the plugin `CookieClicker.jar` file  
2. Place it in your server's `plugins` folder  
3. Restart or reload your server  
4. /cc getclicker to get a new Cookie Block  
5. Place the Block wherever you want  
6. Done!
